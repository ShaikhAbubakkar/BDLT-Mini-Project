# 🏠 House Rental System - Blockchain Based

A decentralized house rental application built on blockchain technology using Solidity smart contracts, React.js frontend, and Polygon network.

## Project Overview

This project implements a transparent, secure, and automated rental system where:
- **Landlords** can list properties and receive rental payments
- **Tenants** can browse properties and sign rental agreements
- **Smart Contracts** automate payments and enforce terms
- **Blockchain** ensures immutable records and transparency
